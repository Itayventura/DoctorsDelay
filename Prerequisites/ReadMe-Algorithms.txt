1. Asumming that python (3.8) is installed.
2. pip should be included in the installation of python and exists in the environment variables.
3. run (double-click or via cmd) the Installation-Script.bat in Prerequisites directory. 
	That will install the prerequisted related to Algorithms.
4. go to config.xml in this project and change the path to the model.pkl file exists in your 
	computer(absulte path).
5. To be able the Algorithms make estimations, the api python should be up. To do so, go to
	Algorithms\scripts and run ApiRunner.bat.
6. Now you are ready to make some predictions :) !
