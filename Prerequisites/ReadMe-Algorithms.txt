1. Asumming that python (3.8) is installed.
2. pip should be included in the installation of python and exists in the environment variables.
3. run (double-click or via cmd) the Installation-Script.bat. That will install the 
	prerequisted related to Algorithms.
4. To be able the Algorithms make estimations, the api python should be up. To to so, go to
	Algorithms\scripts and run ApiRunner.bat.
5. Now you are ready to make some predictions :) !
